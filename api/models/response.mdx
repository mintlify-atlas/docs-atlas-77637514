---
title: ModelResponse
description: Response objects from language models
---

The `ModelResponse` class contains the response from a language model, including content, tool calls, media outputs, and metrics.

## Core Fields

<ParamField path="role" type="str" default="None">
  Role of the response message.
</ParamField>

<ParamField path="content" type="Any" default="None">
  Text content of the response.
</ParamField>

<ParamField path="parsed" type="Any" default="None">
  Parsed structured output (when using output_schema).
</ParamField>

## Media Outputs

<ParamField path="audio" type="Audio" default="None">
  Audio generated by the model.
</ParamField>

<ParamField path="images" type="List[Image]" default="None">
  Images generated by LLM or tools.
</ParamField>

<ParamField path="videos" type="List[Video]" default="None">
  Videos generated by tools.
</ParamField>

<ParamField path="audios" type="List[Audio]" default="None">
  Multiple audio outputs from tools.
</ParamField>

<ParamField path="files" type="List[File]" default="None">
  Files generated by tools.
</ParamField>

## Tool Calls

<ParamField path="tool_calls" type="List[Dict[str, Any]]">
  Model tool calls (function calls to execute).
</ParamField>

<ParamField path="tool_executions" type="List[ToolExecution]">
  Actual tool execution results.
</ParamField>

## Reasoning

<ParamField path="reasoning_content" type="str" default="None">
  Reasoning/thinking content from the model.
</ParamField>

<ParamField path="redacted_reasoning_content" type="str" default="None">
  Redacted reasoning content.
</ParamField>

## Metadata

<ParamField path="event" type="str" default="assistant_response">
  Event type: "assistant_response", "tool_call_started", etc.
</ParamField>

<ParamField path="provider_data" type="Dict[str, Any]" default="None">
  Provider-specific metadata.
</ParamField>

<ParamField path="citations" type="Citations" default="None">
  Citations from the model.
</ParamField>

<ParamField path="response_usage" type="MessageMetrics" default="None">
  Token usage and timing metrics.
</ParamField>

<ParamField path="created_at" type="int">
  Unix timestamp of response creation.
</ParamField>

## State Updates

<ParamField path="updated_session_state" type="Dict[str, Any]" default="None">
  Session state updates from tool executions.
</ParamField>

## Compression

<ParamField path="compression_stats" type="Dict[str, Any]" default="None">
  Statistics about context compression if applied.
</ParamField>

## Model Metrics

<ParamField path="input_tokens" type="int" default="None">
  Number of input tokens.
</ParamField>

<ParamField path="output_tokens" type="int" default="None">
  Number of output tokens.
</ParamField>

<ParamField path="total_tokens" type="int" default="None">
  Total token count.
</ParamField>

<ParamField path="time_to_first_token" type="float" default="None">
  Time to first token in seconds.
</ParamField>

<ParamField path="reasoning_tokens" type="int" default="None">
  Number of reasoning tokens.
</ParamField>

<ParamField path="cache_read_tokens" type="int" default="None">
  Tokens read from cache.
</ParamField>

<ParamField path="cache_write_tokens" type="int" default="None">
  Tokens written to cache.
</ParamField>

## Methods

### to_dict()

Serialize response to dictionary.

```python
data = response.to_dict()
```

### from_dict()

Deserialize response from dictionary.

```python
response = ModelResponse.from_dict(data)
```

## ToolExecution

Represents the execution of a tool.

<ParamField path="tool_call_id" type="str" default="None">
  ID of the tool call.
</ParamField>

<ParamField path="tool_name" type="str" default="None">
  Name of the tool executed.
</ParamField>

<ParamField path="tool_args" type="Dict[str, Any]" default="None">
  Arguments passed to the tool.
</ParamField>

<ParamField path="result" type="str" default="None">
  Result returned by the tool.
</ParamField>

<ParamField path="tool_call_error" type="bool" default="None">
  Whether the tool call resulted in an error.
</ParamField>

<ParamField path="child_run_id" type="str" default="None">
  If the tool created a run (agent/team/workflow), its run ID.
</ParamField>

### HITL Fields

<ParamField path="requires_confirmation" type="bool" default="None">
  Whether the tool requires user confirmation.
</ParamField>

<ParamField path="confirmed" type="bool" default="None">
  Whether the tool has been confirmed.
</ParamField>

<ParamField path="requires_user_input" type="bool" default="None">
  Whether the tool requires user input.
</ParamField>

<ParamField path="external_execution_required" type="bool" default="None">

## Example Usage

<CodeGroup>
```python Basic Response
response = agent.run("What's 2+2?")

print(response.content)  # "The answer is 4."
print(response.response_usage.input_tokens)
print(response.response_usage.output_tokens)
```

```python With Tool Calls
response = agent.run("What's the weather?")

for execution in response.tool_executions:
    print(f"Tool: {execution.tool_name}")
    print(f"Args: {execution.tool_args}")
    print(f"Result: {execution.result}")
```

```python Structured Output
from pydantic import BaseModel

class Answer(BaseModel):
    result: int
    explanation: str

agent = Agent(model="gpt-4o", output_schema=Answer)
response = agent.run("What's 5+7?")

print(response.parsed.result)  # 12
print(response.parsed.explanation)
```

```python With Media
response = agent.run("Generate an image of a sunset")

if response.images:
    for img in response.images:
        print(f"Image: {img.url}")
        img.save("sunset.png")
```
</CodeGroup>
