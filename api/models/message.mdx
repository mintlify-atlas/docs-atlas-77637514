---
title: Message
description: Message objects for agent conversations
---

The `Message` class represents a single message in an agent conversation. It supports text content, tool calls, media attachments, and metadata.

## Constructor

```python
from agno.models.message import Message

message = Message(
    role="user",
    content="Hello!"
)
```

## Core Fields

<ParamField path="role" type="str" required>
  The role of the message author: "system", "user", "assistant", or "tool".
</ParamField>

<ParamField path="content" type="str | List[Any]" default="None">
  The contents of the message. Can be a string or list of content blocks.
</ParamField>

<ParamField path="id" type="str">
  Unique identifier for the message (auto-generated).
</ParamField>

<ParamField path="name" type="str" default="None">
  Optional name for the message participant.
</ParamField>

## Tool Calls

<ParamField path="tool_calls" type="List[Dict[str, Any]]" default="None">
  Tool calls generated by the model.
</ParamField>

<ParamField path="tool_call_id" type="str" default="None">
  Tool call ID that this message is responding to (for tool messages).
</ParamField>

<ParamField path="tool_name" type="str" default="None">
  Name of the tool called.
</ParamField>

<ParamField path="tool_args" type="Any" default="None">
  Arguments passed to the tool.
</ParamField>

<ParamField path="tool_call_error" type="bool" default="None">
  Whether the tool call resulted in an error.
</ParamField>

## Media

<ParamField path="images" type="Sequence[Image]" default="None">
  Images attached to the message.
</ParamField>

<ParamField path="audio" type="Sequence[Audio]" default="None">
  Audio files attached to the message.
</ParamField>

<ParamField path="videos" type="Sequence[Video]" default="None">
  Videos attached to the message.
</ParamField>

<ParamField path="files" type="Sequence[File]" default="None">
  Files attached to the message.
</ParamField>

## Output Media

<ParamField path="audio_output" type="Audio" default="None">
  Audio generated by the model.
</ParamField>

<ParamField path="image_output" type="Image" default="None">
  Image generated by the model.
</ParamField>

<ParamField path="video_output" type="Video" default="None">
  Video generated by the model.
</ParamField>

## Reasoning

<ParamField path="reasoning_content" type="str" default="None">
  The reasoning/thinking content from the model (not sent to API).
</ParamField>

<ParamField path="redacted_reasoning_content" type="str" default="None">
  Redacted reasoning content sent to the model API.
</ParamField>

## Metadata

<ParamField path="metrics" type="MessageMetrics">
  Metrics for the message (tokens, timing, etc.).
</ParamField>

<ParamField path="references" type="MessageReferences" default="None">
  References added to the message for RAG.
</ParamField>

<ParamField path="citations" type="Citations" default="None">
  Citations received from the model.
</ParamField>

<ParamField path="provider_data" type="Dict[str, Any]" default="None">
  Provider-specific data needed for subsequent messages.
</ParamField>

<ParamField path="created_at" type="int">
  Unix timestamp when the message was created.
</ParamField>

## Flags

<ParamField path="stop_after_tool_call" type="bool" default="False">
  If True, the agent stops executing after this tool call.
</ParamField>

<ParamField path="add_to_agent_memory" type="bool" default="True">
  Whether to add this message to the agent's memory.
</ParamField>

<ParamField path="from_history" type="bool" default="False">
  Flag indicating the message was fetched from history.
</ParamField>

<ParamField path="temporary" type="bool" default="False">
  If True, the message is sent to the model but not persisted.
</ParamField>

## Methods

### get_content_string()

Return content as a string.

```python
text = message.get_content_string()
```

### to_dict()

Serialize message to dictionary.

```python
data = message.to_dict()
```

### from_dict()

Deserialize message from dictionary.

```python
message = Message.from_dict(data)
```

### log()

Log the message to console.

```python
message.log(metrics=True, level="info")
```

## Example Usage

<CodeGroup>
```python User Message
from agno.models.message import Message

message = Message(
    role="user",
    content="What's the weather?"
)
```

```python With Images
from agno.models.message import Message
from agno.media import Image

message = Message(
    role="user",
    content="Describe this image",
    images=[Image(url="https://example.com/image.jpg")]
)
```

```python Tool Response
from agno.models.message import Message

message = Message(
    role="tool",
    content="Tokyo: 72°F, sunny",
    tool_call_id="call_123",
    tool_name="get_weather"
)
```

```python With Reasoning
from agno.models.message import Message

message = Message(
    role="assistant",
    reasoning_content="Let me think through this step by step...",
    content="The answer is 42."
)
```
</CodeGroup>
