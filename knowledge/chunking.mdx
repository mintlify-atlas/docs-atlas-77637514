---
title: "Document chunking"
description: "Split documents into chunks for embedding"
---

Chunking divides documents into smaller pieces for better retrieval.

## Default chunking

```python
knowledge = PDFKnowledge(
    path="docs/",
    vector_db=vector_db,
    chunk_size=1000,  # Characters per chunk
    chunk_overlap=200  # Overlap between chunks
)
```

## Semantic chunking

Use Chonkie for semantic-aware chunking:

```python
from agno.knowledge.pdf import PDFKnowledge
from chonkie import SemanticChunker

knowledge = PDFKnowledge(
    path="docs/",
    vector_db=vector_db,
    chunking_strategy=SemanticChunker()
)
```

See [cookbook examples](/cookbook/agents) for patterns.
